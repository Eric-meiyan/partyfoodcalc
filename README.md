# 🎉 Party Food Calculator

**[PartyFoodCalc.com](https://partyfoodcalc.com)** — Calculate exactly how much food you need for any party. Never run out or waste money again!

## Features

- 🧮 **Smart Calculator** — Enter guests, party type, and duration for instant food recommendations
- 🍗 **10+ Food Calculators** — Specialized calculators for chicken wings, pizza, tacos, BBQ, burgers, and more
- 🛒 **Shopping List** — Generate a printable/copyable shopping list with exact quantities
- 🍺 **Drinks Calculator** — Beer, wine, cocktails, and non-alcoholic beverages
- 💒 **Event-Specific** — Wedding food calculator, Super Bowl party calculator
- 📱 **Mobile-Friendly** — Works perfectly on any device
- ⚡ **No Signup Required** — 100% free, no account needed

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Analytics:** Google Analytics 4
- **SEO:** next-sitemap, structured data (Schema.org)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
